# Facial Beauty Prediction System

This project implements a facial beauty prediction system using MediaPipe Face Mesh and Machine Learning (Linear Regression).

## Features
- Facial landmark detection (468 points)
- Symmetry & proportion feature extraction
- Rule-based vs Machine Learning comparison
- Web-based UI (upload & real-time camera)

## Model
- Linear Regression (Scikit-learn)
- Evaluation: Mean Absolute Error (MAE)

## Dataset
- SCUT-FBP5500

## How to Run
pip install -r requirements.txt
python main.py
